Para compilar e executar o EP, é necessário executar os seguintes passos:

->compilar o arquivo ep.c, gerando o arquivo executável ep
$ gcc ep.c -o ep

->executar o arquivo ep passando dois paramêtros o arquivo a ser
 lido e a estrutura a ser usada(lista ou árvore)
$ ./ep texto.txt arvore

As instruções para utilizar o programa são as dadas no arquivo de enunciado

*O programa possui um tamanho máximo de linha. Caso haja erro na execução,
quebre as linhas do texto para que tenham no máximo 100 caracteres
